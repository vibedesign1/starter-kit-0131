```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│   온보딩    │ ──▶ │    홈       │ ──▶ │  전시 만들기 │
└─────────────┘     └─────────────┘     └─────────────┘
                          │                    │
                          ▼                    ▼
                    ┌─────────────┐     ┌─────────────┐

                    └─────────────┘
```

## 2. 온보딩 플로우

### 2.1 플로우 다이어그램

```
[앱 시작] 
    │
    ▼
[스플래시 화면]
    │
    ▼
[온보딩 슬라이드 (3장)]
    │ 스킵 가능
    ▼
[로그인/회원가입 선택]
    │
    ├──▶ [소셜 로그인] ──▶ [프로필 설정] ──▶ [홈]
    │
    └──▶ [둘러보기] ──▶ [홈 (제한된 기능)]
```

### 2.2 화면별 상세

| 화면 | 사용자 액션 | 시스템 반응 |
|------|------------|------------|
| 스플래시 | 대기 | 2초 후 자동 전환 |
| 온보딩 1 | 스와이프/다음 | 서비스 소개 표시 |
| 온보딩 2 | 스와이프/다음 | 주요 기능 설명 |
| 온보딩 3 | 시작하기 탭 | 로그인 화면 이동 |
| 로그인 | 소셜 로그인 선택 | OAuth 인증 진행 |
| 프로필 설정 | 닉네임 입력, 완료 | 홈 화면 이동 |

## 3. 홈 화면 플로우

### 3.1 플로우 다이어그램

```
[홈 화면]
    │
    ├──▶ [+ 새 전시 만들기] ──▶ 전시 만들기 플로우
    │
    ├──▶ [나의 전시 목록]
    │         │
    │         ├──▶ [전시 선택] ──▶ 전시 보기 플로우
    │         │
    │         └──▶ [전시 편집] ──▶ 전시 편집 플로우
    │
    ├──▶ [설정]
    │
    └──▶ [프로필]
```

### 3.2 화면 구성 요소

| 영역 | 구성 요소 | 동작 |
|------|----------|------|
| 상단 | 로고, 프로필 아이콘 | 프로필 페이지 이동 |
| 메인 | 나의 전시 카드 목록 | 전시 선택 시 보기 모드 |
| 플로팅 | + 버튼 | 새 전시 만들기 |
| 하단 | 네비게이션 바 | 홈/탐색/설정 전환 |

## 4. 전시 만들기 플로우

### 4.1 전체 플로우 다이어그램

```
[새 전시 만들기]
    │
    ▼
[STEP 1: 전시 기본 정보]
    │ 전시 제목, 소개글, 테마 선택
    ▼
[STEP 2: 책 추가]
    │
    ├──▶ [책 검색] ──▶ [검색 결과] ──▶ [책 선택]
    │                                      │
    ├──▶ [바코드 스캔] ──▶ [책 정보 확인] ──┤
    │                                      │
    └──▶ [직접 입력] ──────────────────────┘
                                           │
                                           ▼
                                    [책 상세 정보 입력]
                                           │ 코멘트, 추억, 문장 등
                                           ▼
                                    [책 추가 완료]
                                           │
                                           ▼
                                    [책 목록으로 복귀]
                                           │ 추가 책 등록 또는 다음
                                           ▼
[STEP 3: 전시 구성]
    │ 책 순서 배치, 섹션 구분
    ▼
[STEP 4: 미리보기]
    │
    ├──▶ [수정하기] ──▶ 해당 단계로 이동
    │
    └──▶ [전시 완료] ──▶ [완료 화면] ──▶ [홈]
```

### 4.2 STEP 1: 전시 기본 정보

```
┌────────────────────────────────────┐
│  ← 새 전시 만들기          [1/4]   │
├────────────────────────────────────┤
│                                    │
│  전시 제목 *                       │
│  ┌────────────────────────────┐   │
│  │ 예: 나를 만든 책들          │   │
│  └────────────────────────────┘   │
│                                    │
│  전시 소개글                       │
│  ┌────────────────────────────┐   │
│  │                            │   │
│  │                            │   │
│  └────────────────────────────┘   │
│                                    │
│  테마 선택                         │
│  ┌──────┐ ┌──────┐ ┌──────┐      │
│  │클래식│ │ 모던 │ │ 서재 │      │
│  └──────┘ └──────┘ └──────┘      │
│                                    │
├────────────────────────────────────┤
│         [ 다음 단계 ]              │
└────────────────────────────────────┘
```

| 입력 필드 | 필수 여부 | 유효성 검사 |
|----------|----------|------------|
| 전시 제목 | 필수 | 2-30자 |
| 전시 소개글 | 선택 | 최대 500자 |
| 테마 선택 | 필수 | 기본값: 클래식 |

### 4.3 STEP 2: 책 추가

#### 4.3.1 책 검색 플로우

```
[책 추가 화면]
    │
    ▼
┌────────────────────────────────────┐
│  ← 책 추가                         │
├────────────────────────────────────┤
│  ┌────────────────────────────┐   │
│  │ 🔍 책 제목 또는 저자 검색   │   │
│  └────────────────────────────┘   │
│                                    │
│  [바코드 스캔]  [직접 입력]        │
│                                    │
│  ─────────────────────────────    │
│  검색 결과                         │
│  ┌────────────────────────────┐   │
│  │ 📚 책제목 1                │   │
│  │    저자명 | 출판사         │   │
│  └────────────────────────────┘   │
│  ┌────────────────────────────┐   │
│  │ 📚 책제목 2                │   │
│  │    저자명 | 출판사         │   │
│  └────────────────────────────┘   │
└────────────────────────────────────┘
```

#### 4.3.2 책 상세 정보 입력 플로우

```
[책 선택]
    │
    ▼
┌────────────────────────────────────┐
│  ← 책 정보 입력                    │
├────────────────────────────────────┤
│        ┌──────────┐                │
│        │  📖      │                │
│        │ 책 표지  │                │
│        └──────────┘                │
│        책 제목                      │
│        저자 | 출판사                │
│                                    │
│  ─────────────────────────────    │
│                                    │
│  나의 코멘트 *                     │
│  ┌────────────────────────────┐   │
│  │ 이 책을 읽고 느낀 점...     │   │
│  └────────────────────────────┘   │
│                                    │
│  추억 스토리                       │
│  ┌────────────────────────────┐   │
│  │ 이 책과 관련된 추억...      │   │
│  └────────────────────────────┘   │
│                                    │
│  [더 많은 정보 입력 ▼]             │
│                                    │
├────────────────────────────────────┤
│         [ 책 추가하기 ]            │
└────────────────────────────────────┘
```

#### 4.3.3 확장 입력 필드

| 필드 | 입력 타입 | 설명 |
|------|----------|------|
| 읽은 시기 | 날짜 선택 | 연/월 또는 대략적 시기 |
| 읽은 장소 | 텍스트 | 예: 동네 도서관, 제주도 카페 |
| 함께한 사람 | 텍스트 | 예: 어머니, 친구 OOO |
| 기억에 남는 문장 | 텍스트 | 책에서 인용 |
| 별점 | 1-5 선택 | 별 아이콘 탭 |
| 태그 | 다중 선택/입력 | 예: #감동, #성장, #추천 |

### 4.4 STEP 3: 전시 구성

```
┌────────────────────────────────────┐
│  ← 전시 구성               [3/4]   │
├────────────────────────────────────┤
│                                    │
│  배치 방식 선택                    │
│  ○ 시간순  ○ 추천순  ● 직접 배치  │
│                                    │
│  ─────────────────────────────    │
│  드래그하여 순서를 변경하세요      │
│                                    │
│  ┌─────┐ ┌─────┐ ┌─────┐         │
│  │ 1   │ │ 2   │ │ 3   │         │
│  │책표지│ │책표지│ │책표지│         │
│  └─────┘ └─────┘ └─────┘         │
│                                    │
│  ┌─────┐ ┌─────┐                  │
│  │ 4   │ │ 5   │                  │
│  │책표지│ │책표지│                  │
│  └─────┘ └─────┘                  │
│                                    │
│  [+ 섹션 추가]                     │
│                                    │
├────────────────────────────────────┤
│         [ 미리보기 ]               │
└────────────────────────────────────┘
```

| 기능 | 동작 | 피드백 |
|------|------|--------|
| 드래그 앤 드롭 | 책 카드 길게 누르기 | 진동 + 카드 확대 |
| 순서 변경 | 드래그하여 위치 이동 | 실시간 재배치 |
| 섹션 추가 | 구분선 삽입 | 섹션 제목 입력 팝업 |

### 4.5 STEP 4: 미리보기

```
[미리보기 화면]
    │
    ├──▶ [복도 뷰 미리보기]
    │         │
    │         └──▶ 좌우 스크롤로 전시 체험
    │
    ├──▶ [책 탭] ──▶ [큐레이터 뷰 미리보기]
    │
    ├──▶ [수정하기] ──▶ 해당 단계로 이동
    │
    └──▶ [전시 완료하기]
              │
              ▼
        [완료 팝업]
              │
              ├──▶ [바로 공유하기] ──▶ 공유 플로우
              │
              └──▶ [홈으로] ──▶ 홈 화면
```

## 5. 전시 보기 플로우

### 5.1 전체 플로우 다이어그램

```
[전시 선택]
    │
    ▼
[전시 입장 화면]
    │ 전시 제목, 소개, 입장하기 버튼
    ▼
[복도 뷰 (Gallery Walk)]
    │
    ├──▶ [좌우 스크롤] ──▶ 다음/이전 책으로 이동
    │
    ├──▶ [책 탭] ──▶ [큐레이터 뷰 (Detail View)]
    │                      │
    │                      ├──▶ [이전/다음 책]
    │                      │
    │                      └──▶ [복도 뷰로 돌아가기]
    │
    ├──▶ [공유 버튼] ──▶ 공유 플로우
    │
    └──▶ [나가기] ──▶ 홈 화면
```

## 8. 에러 및 예외 처리 플로우

```
[API 요청 실패]
    │
    ▼
[에러 화면/토스트]
    │
    ├──▶ [다시 시도] ──▶ 재요청
    │
    └──▶ [오프라인 모드] ──▶ 캐시된 데이터 표시
```

### 8.2 책 검색 결과 없음

```
[검색 결과 없음]
    │
    ▼
┌────────────────────────────────────┐
│                                    │
│        검색 결과가 없습니다        │
│                                    │
│   [ 직접 입력하기 ]                │
│                                    │
└────────────────────────────────────┘
```

### 8.3 에러 상황별 처리

| 네트워크 끊김 | 토스트 메시지 | 다시 시도 버튼 |

| 이미지 로드 실패 | 플레이스홀더 | 기본 책 이미지 |
| 저장 실패 | 에러 팝업 | 임시 저장 안내 |

## 9. 상태별 화면 정의

### 9.1 홈 화면 상태

| 상태 | 화면 구성 |
|------|----------|
| 전시 없음 (Empty) | 일러스트 + "첫 전시를 만들어보세요" + CTA 버튼 |
| 전시 있음 | 전시 카드 그리드/리스트 |
| 로딩 중 | 스켈레톤 UI |
| 에러 | 에러 메시지 + 새로고침 버튼 |

### 9.2 전시 만들기 상태

| 상태 | 화면 구성 |
|------|----------|
| 책 0권 | "책을 추가해주세요" 안내 |
| 책 1권 이상 | 책 목록 + 추가 버튼 |
| 저장 중 | 로딩 인디케이터 |
| 저장 완료 | 성공 애니메이션 + 완료 화면 |
