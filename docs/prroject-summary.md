- 나의 추억 전시관 

  - 개요 : 

    - 자녀와 어린 시절 도서관에서 같이 앉아 읽었던 책들이 있었다. 
    - 사춘기 시절 우연히 형제가 읽던 책의 제목이 우연히 눈에 들어와 읽었는데 지금도 그 책을 읽었던 느낌을 잊을 수 없다.
    - 서점에서 돌아 다니다 만나는 베스트셀러 중 하나가 손에 잡혀 읽어 보았다. 지인들에게도 소개하고자 한다. 
    - 이런 책과 관련된 좋은 기억들을 "나의 전시" 을 통해 표현해 보고자 한다.

  - 기능 : 

    - 나의 전시 만들기

      - 전시에 사용할 책 들 표시
      - 나의 전시에 사용할 책 선택하여 나의 전시 생성

    - 나의 전시 보기 

      - 1) 전시 책 목록 : 책 표지 + 간단한 설명 + 나의 코멘트

        - cf) 그림 전시할 때 벽에 그림 + 설명택 이 있는 복도를 걸어가는 느낌.

      - 2) 전시 책 선택 : 

        - 책 하나를 기준으로 좀 더 다양한 내용을 이용하여 책에 대하여 설명
        - cf) 큐레이터가 하나의 유물에 대하여 설명할 때 느낌.

  - 문의 : 

    - 전시에 필요한 책 정보 항목은 무었인가. 항목을 제시하라.
    - 전시의 방식이나 구성에 대한 아이디어를 제시하라.
