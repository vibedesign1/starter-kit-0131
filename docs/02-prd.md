---

## 1. 개요

"나의 추억 전시관"은 사용자가 책과 관련된 소중한 기억들을 디지털 전시 형태로 기록하고 공유할 수 있는 서비스입니다. 자녀와 함께 도서관에서 읽었던 책, 사춘기 시절 우연히 만난 인생 책, 서점에서 발견한 베스트셀러 등 책과 연결된 추억을 "나의 전시"라는 형식으로 표현합니다.

---

## 2. 제품 비전 및 목표

### 2.1 비전

모든 사람이 자신만의 책 관련 추억을 아름답게 전시하고, 소중한 사람들과 나눌 수 있는 개인 디지털 전시 플랫폼

### 2.2 목표

- 책과 관련된 개인적인 추억을 체계적으로 기록할 수 있는 환경 제공
- 실제 미술관/박물관을 방문하는 듯한 몰입감 있는 전시 경험 제공
- 소셜 기능을 통해 지인들과 책 추천 및 추억 공유 활성화

---

## 3. 핵심 기능

### 3.1 나의 전시 만들기

#### 3.1.1 책 검색 및 선택

- 도서 검색 API 연동을 통한 책 정보 자동 입력
- ISBN 바코드 스캔 기능 (모바일)
- 수동 입력 옵션 (검색되지 않는 책의 경우)

#### 3.1.2 전시 구성

- 전시 제목 및 소개글 작성
- 책 순서 배치 (드래그 앤 드롭)
- 각 책에 대한 개인 코멘트 작성
- 전시 테마/배경 선택

### 3.2 나의 전시 보기

#### 3.2.1 복도 뷰 (Gallery Walk)

미술관 복도를 걸어가며 그림을 감상하는 느낌의 UI/UX를 제공합니다.

- 책 표지 이미지 + 간단한 설명 태그 표시
- 나의 코멘트 일부 미리보기
- 좌우 스크롤 또는 터치 제스처로 이동

#### 3.2.2 큐레이터 뷰 (Detail View)

큐레이터가 유물을 상세히 설명하는 느낌의 상세 페이지를 제공합니다.

- 책의 전체 정보 표시
- 나의 상세 코멘트 및 추억 스토리
- 관련 인용구 또는 기억에 남는 문장
- 읽은 날짜, 장소 등 맥락 정보

## 4. 책 정보 항목 (Data Schema)

전시에 필요한 책 정보 항목은 다음과 같이 구성됩니다.

### 4.1 기본 정보 (자동 수집)

| 항목 | 설명 | 비고 |
|------|------|------|
| ISBN | 국제표준도서번호 | 고유 식별자 |
| 제목 | 책 제목 | 필수 |
| 저자 | 저자명 (복수 가능) | 필수 |
| 출판사 | 출판사명 | 선택 |
| 출판일 | 초판 발행일 | 선택 |
| 표지 이미지 | 책 표지 이미지 URL | 필수 |
| 장르/카테고리 | 도서 분류 | 선택 |
| 페이지 수 | 총 페이지 수 | 선택 |
| 책 소개 | 출판사 제공 소개글 | 선택 |

### 4.2 개인 기록 정보 (사용자 입력)

| 항목 | 설명 | 비고 |
|------|------|------|
| 나의 코멘트 | 책에 대한 개인적인 감상 | 필수 |
| 추억 스토리 | 책과 관련된 추억 이야기 | 선택 |
| 읽은 시기 | 책을 읽은 대략적인 시기 | 선택 |
| 읽은 장소 | 책을 읽었던 장소 | 선택 |
| 함께한 사람 | 함께 읽거나 추천받은 사람 | 선택 |
| 기억에 남는 문장 | 인상 깊은 구절 인용 | 선택 |
| 별점 | 개인 평점 (1-5) | 선택 |
| 태그 | 개인 분류용 태그 | 선택 |

## 5. 전시 방식 및 구성 아이디어

### 5.1 전시 테마 옵션

- **클래식 갤러리**: 고전적인 미술관 분위기 (흰 벽, 나무 프레임)
- **모던 뮤지엄**: 현대적이고 미니멀한 디자인
- **아늑한 서재**: 따뜻한 조명과 나무 책장 느낌
- **계절 테마**: 봄/여름/가을/겨울 분위기 전환

### 5.2 전시 구성 방식

- **시간순 배치**: 읽은 순서대로 타임라인 형태
- **테마별 섹션**: 장르, 감정, 상황별 그룹핑
- **추천순 배치**: 가장 추천하고 싶은 책부터 배열
- **사용자 정의 순서**: 자유롭게 배치

### 5.3 인터랙션 아이디어

- **오디오 가이드**: 내 코멘트를 음성으로 재생
- **방명록 기능**: 전시를 본 사람들의 한 줄 감상
- **공유 링크**: 지인에게 전시 초대 가능
- **QR 코드 생성**: 오프라인에서도 전시 공유

## 6. 사용자 시나리오

### 시나리오 1: 자녀와의 추억 전시

사용자 A는 어린 자녀와 도서관에서 함께 읽었던 그림책 10권을 모아 "우리 아이와 함께 읽은 책" 전시를 만들고자 합니다. 각 책마다 아이의 반응과 함께 읽을 때의 에피소드를 기록하여, 아이가 성장한 후 함께 돌아볼 수 있는 디지털 앨범을 만듭니다.

### 시나리오 2: 인생 책 공유

사용자 B는 사춘기 시절 형의 책장에서 우연히 발견한 책이 인생을 바꿨습니다. 그 책을 중심으로 "나를 만든 책들" 전시를 구성하여 친구들에게 공유합니다. 전시를 본 친구들이 방명록에 자신의 인생 책을 남기며 소통합니다.

## 7. 기술 요구사항

- 도서 정보 API 연동 (네이버 책, 국립중앙도서관 API 등)
- 반응형 웹/모바일 앱 지원
- 이미지 최적화 및 CDN 활용
- 소셜 로그인 및 공유 기능
- 개인정보 보호 및 전시 공개 범위 설정
