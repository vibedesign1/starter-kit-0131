0. 배경 & 문제정의

현대 주거 공간에서 조명은 단순한 스위치가 아니라, 하루의 리듬(시간·환경)에 반응해 시력을 보호하고 무드를 조율해야 함. Lumenstate는 “빛은 공간의 상태다”라는 브랜드 신조 아래, 환경 반응형 조도/색온도 커브를 제안한다.

1. 목표(Goals)

브랜드 메시지(철학)와 핵심 기능(환경 반응형 라이팅)을 명료하게 전달한다.

제품 썸네일을 정면·평면·여백 중심 아트디렉션으로 일관되게 노출한다.

사용자에게 낮/저녁/밤 시나리오를 직관적으로 체험시키고, 조도·색온도 개념을 이해시키는 간단한 인터랙션을 제공한다.

2. 비범위(Non-Goals)

커머스 결제/장바구니, 설치 상담 예약 등 트랜잭션 기능

상세 구성요소(앱·디바이스 설정) 편집 UI

실시간 센서 연동(파일럿 단계에서 제외)

3. 타깃 사용자(Persona 요약)

홈 라이팅에 민감한 거주자(서재/식탁/침실의 눈 피로·분위기 개선 니즈)

디자인/브랜딩 민감 사용자(미니멀 미감, 정면 아트 디렉션 선호)

4. 핵심 가치 제안

“빛은 공간의 상태다.” 하루의 곡선을 따라 조도·색온도를 부드럽게 조율.

시력 보호·수면 위생·대화 품질 향상을 위한 장면 프리셋 제공(개념 소개 수준)

5. IA / 섹션 범위

5.1 히어로

헤드라인: 빛은 공간의 상태다. (Light is the state of space.)

서브: 하루의 곡선을 따라 조도·색온도를 부드럽게 조율합니다.

비주얼: 정면·평면 배경 제품 이미지 1장 + 3단계 크로스페이드 루프(낮→저녁→밤). 점멸 없음.

CTA: 제품 보기 / 하루 리듬 시연

모션 원칙: easeInOutSine, 600·900·1200ms (감속/가속 과장 금지)

접근성: prefers-reduced-motion 존중(애니메이션 최소화)

5.2 제품 쇼케이스(그리드 + 시간대 반영)

그리드: Desktop 3×2 / Mobile 2열

카드 구성(최소): 썸네일(정면·평면) / 제품명 / 마감 태그 / 상태 라벨(lux·K)

컨트롤: 타임라인 스크러버(연속 값) — 낮·저녁·밤 마커를 가진 0–1 구간 슬라이더로 모션 영상의 타임라인을 직접 조절(이미지 교체 금지)

전환: 스크러버 이동 시 영상의 프레임 시킹(seek) 및 조도·색온도 실시간 업데이트(즉시 반응, crossfade 없음)

자산 전략: 제품별 **단일 모션 소스(비디오/스프라이트/Canvas WebGL)**를 제공하여 타임라인 시킹으로 상태 표현(3컷 정적 이미지 대체)

6. 컬러 스킴(총 4색)

사이트 전반은 아래 4색만 사용. 그래디언트/글로우/블러 금지.

Wall Tint White — 실제 벽 느낌의 틴트된 하양(라이트 배경)

Hex: #F5F2EE

3800K White — 3800K에 맞춘 따뜻한 백색(라이트 보조/다크 텍스트)

Hex: #F2E9DA

Warm Black — 3800K 스펙트럼 계열의 따뜻한 흑(다크 배경)

Hex: #12100E

3800K Accent — 하이라이트(링크/CTA/포커스)

Hex: #FFC66E

라이트 모드: 배경 #F5F2EE / 본문 #12100E / 보조텍스트 #12100E 80% / 액센트 #FFC66E다크 모드: 배경 #12100E / 본문 #F2E9DA / 보조텍스트 #F2E9DA 80% / 액센트 #FFC66E

접근성: 본문 대비 WCAG AA 이상 확보. 액센트 텍스트는 면적/상태에 한정(가독 우선).

8. 기능 요구사항(Functional)

[F-1] 히어로 3단계 루프 모션(낮/저녁/밤) — 이미지 간 크로스페이드, 점멸 금지

[F-2] 쇼케이스 타임라인 스크러버 — 사용자가 슬라이더를 움직이면 모션 소스 타임라인을 시킹하고 상태 라벨(lux·K)을 실시간 갱신(이미지 교체 방식 금지)

[F-3] Lazy-load — 카드 썸네일 지연 로딩, 3컷 사전 프리로드

[F-4] 다크모드 지원 — 배경/텍스트만 4색 스킴 내에서 전환
