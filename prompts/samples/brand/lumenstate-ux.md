# Lumenstate UX Flow

Lumenstate 브랜드 UX 플로우 참조 문서.

## Page Structure

- Header
- Hero Section
- Brand Value Section
- Product Showcase Section

---

## 1. Header

사용자가 페이지 어디에 있든 상단에 고정 노출

- 로고: 브랜드 인지
- 실시간 시계: 현재 시간 표시, 낮/밤 아이콘으로 시간대 인지
- 타임라인 아이콘: 스크롤 후 표시. 시간대 빠른 전환 제공

---

## 2. Hero Section

첫 화면. 브랜드 메시지와 제품 분위기 전달

**정보 전달 순서:**
1. 브랜드명 "Lumenstate"
2. 태그라인 "Light defines the space."
3. 배경 비디오로 조명이 공간에 미치는 영향 시각화

**인터랙션:**
- 스크롤 시 비디오가 낮→저녁→밤 장면으로 전환
- 스크롤 시 화면 하단에 현재 시간(12:00pm→12:00am) 표시

---

## 3. Brand Value Section

브랜드의 3가지 핵심 가치 전달

**정보 전달 순서:**
1. Immanence - 공간에 스며드는 빛
2. Continuity - 자연스러운 낮→밤 전환
3. Flexibility - 자동화와 수동 제어의 균형

**인터랙션:** 없음 (정적 콘텐츠)

---

## 4. Product Showcase Section

제품 탐색 및 시간대별 조명 변화 체험

**정보 전달 순서:**
1. 섹션 타이틀 "Product Showcase"
2. 타임라인 슬라이더 (시간대 선택 UI)
3. 제품 필터 (타입별 분류)
4. 제품 그리드

### 시간대별 테마 전환

| 시간 | 배경 | 텍스트 | 제품 사진 |
|------|------|--------|----------|
| 12:00pm (정오) | 밝은 배경 | 어두운 텍스트 | 낮 버전 |
| 4:00pm (오후) | 밝은 배경 유지 | - | - |
| 8:00pm (저녁) | 어두운 배경 | 밝은 텍스트 | 밤 버전 |
| 12:00am (자정) | 어두운 배경 유지 | - | - |

**인터랙션:**
- 타임라인 슬라이더 조작 시 제품 사진이 낮→밤 버전으로 크로스페이드
- 타임라인 슬라이더 조작 시 페이지 배경색이 밝은색→어두운색으로 전환
- 타임라인 슬라이더 조작 시 텍스트 색상이 배경에 맞춰 반전
- 타임라인 아이콘 클릭 시 해당 시간대로 즉시 이동
- 필터 클릭 시 선택한 타입의 제품만 표시
- 슬라이더가 화면 밖으로 스크롤되면 헤더에 미니 타임라인 표시

---

## User Journey

1. 진입
2. 브랜드 인지 (Hero - 로고, 태그라인)
3. 제품 분위기 체험 (Hero - 스크롤로 비디오 낮→밤 전환)
4. 브랜드 철학 이해 (Brand Value - 3가지 핵심 가치)
5. 제품 탐색 (Product Showcase - 필터로 카테고리 선택)
6. 시간대별 변화 체험 (Product Showcase - 슬라이더로 낮/밤 전환)

---

## Key Interactions

| 인터랙션 | 영역 | 동작 |
|---------|------|------|
| 스크롤 | Hero | 비디오 프레임이 시간 흐름에 따라 변화 |
| 타임라인 드래그 | Product Showcase | 제품 사진 + 배경색 + 텍스트색 동시 전환 |
| 타임라인 아이콘 클릭 | Product Showcase | 4단계 시간대 중 하나로 즉시 이동 |
| 필터 선택 | Product Showcase | 제품 그리드 필터링 |
