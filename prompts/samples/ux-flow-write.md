## 작성 가이드

### 섹션 설명
- 각 섹션의 역할과 목적을 **한 줄**로 명확히 설명
- 사용자 관점에서 "무엇을 경험하는가"에 초점

### 정보 전달 순서
- 사용자가 **시각적으로 인지하는 순서**대로 나열
- 위계(hierarchy)와 시간 흐름을 고려

### 인터랙션
- **사용자 액션** → **시각적 변화**로 서술
- 조건부 동작이 있으면 명확히 구분
- 인터랙션이 없으면 "없음 (정적 콘텐츠)" 명시

### User Journey
- 사용자가 페이지를 처음부터 끝까지 경험하는 **감정적/인지적 흐름**
- 단순 섹션 나열이 아닌 **경험의 연속성** 강조

### Key Interactions
- 페이지 전체의 주요 인터랙션을 **한눈에** 파악할 수 있도록 정리
- 중복 제거, 핵심만 선별
